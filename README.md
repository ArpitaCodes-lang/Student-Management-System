# Student Management System

A simple Java console-based Student Management System developed using Object-Oriented Programming (OOP) concepts.

## Features
- Add Student
- View All Students
- Store Student ID, Name, Branch, and CGPA
- Menu-driven console interface

## Technologies Used
- Java
- IntelliJ IDEA
- Git
- GitHub

## Concepts Used
- Classes and Objects
- Constructors
- ArrayList
- Encapsulation
- Loops
- Switch Case
- Scanner

## Project Structure
```
src/
└── com/arpita/sms/
    ├── Main.java
    ├── Student.java
    └── StudentManager.java
```
